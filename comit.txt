подготовленная сборка галпа - база от фрилансер по жизни + webpack

как работать:

Установлен core - js для оптимизации подключаем нужные для работы конкретные полифилы директивой import - это совместимость с es5 и ie11

установлен webpack - есть переключение режимов прода и дева isDev true = development, false = production c минификацией

установка пакетов локально npm i --save-dev name

запуск программ может быть по частям кмандой gulp name

npm cache clean --force (очистка npm)
npm i npm -g (установка npm)


запуск нового проекта

Копируем фаилы в папку старта проекта #src, далее в терминале из папки запускаем npm i для установки пакетов, список пакетов в pacageJson

Добавление фаилов в основную сборку идет дерективой @@include("name") для html

import для scss и js (штатный)

дополнительные фаилы html пишем через _ - _name.html - тогда они не дублируются а собираются в один фаилик в index.html подключаем их через 
 @@include("_header.html")

Очистка папки с картинками gulp cleanimg - вызов отдельно
